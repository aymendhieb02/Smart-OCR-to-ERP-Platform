# Screenshot Checklist

Automated screenshots can be captured from the local browser, but the final presentation should include these exact screens:

1. Landing/upload page with project summary and demo buttons.
2. Review workspace with a document preview loaded.
3. Invoice preview with OCR/layout/field/line overlays enabled.
4. Selected region inspector showing concise evidence and advanced evidence collapsed.
5. Product Lines tab with editable rows, restore, and delete actions.
6. Financial Checks tab showing passed/warning/conflict cards.
7. ERP readiness panel showing export enabled for a valid document.
8. ERP readiness panel showing export blocked for a noisy or uncertain document.
9. ERP JSON tab after correction and revalidation.

Recommended manual naming:

- `landing_upload.png`
- `review_workspace_overlays.png`
- `selected_region_evidence.png`
- `editable_product_table.png`
- `financial_validation.png`
- `erp_ready.png`
- `export_blocked.png`
- `erp_json.png`
